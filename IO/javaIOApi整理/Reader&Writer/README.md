# Reader & Writer

<br>

`InputStream` 與 `OutputStream` 是用來讀寫位元組資料。如果實際上要在 java 應用中處理字元資料，使用 `InputStream` 與 `OutputStream` 還要對照編碼表。在字元與位元組之間轉換。

java SE API 已有提供對應的字元 IO 處理類別，簡化這類的操作。這便是本篇重點 `Reader` & `Writer`
