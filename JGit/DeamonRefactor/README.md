## 說明

前一個 clone git repo 實作的那個計時器的方法不是很好，研究了一點點執行緒觀念之後，發現計時器在這個小項目上的應用完全符合 Deamon 存在的意義，守護進程，在主進程結束時自己就無條件結束。這邊對應到這個小項目中，就是當 clone repo 完成時計時任務就直接結束。

這裏其實還有一個應用方式是使用 FutureTask，就留給以後有空再實作吧！
